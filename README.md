逆離散フーリエ変換（IDFT）を用いたOFDM（Orthogonal Frequency Division Multiplexing：直交周波数分割多重）のサブキャリア生成を体感できる教育的なデモアプリケーション

## 概要

このデモでは、OFDMの基本原理である「周波数領域での複数の直交サブキャリアへのデータ割り当て」と「IDFTによる時間領域信号への変換」を可視化します。

### OFDMとは

OFDM（直交周波数分割多重）は、高速無線通信で広く使用される変調方式です：

- **Wi-Fi (IEEE 802.11a/g/n/ac/ax)**
- **LTE/5G移動通信**
- **地上デジタル放送（ISDB-T）**
- **ADSL**

OFDMでは、データストリームを複数の低速な並列ストリームに分割し、それぞれを異なる周波数のサブキャリアで伝送します。各サブキャリアは互いに直交しているため、周波数帯域を効率的に利用しながら、マルチパス干渉に強い通信が可能です。

## 機能

- **サブキャリア数の調整**: 4〜16個のサブキャリアを設定可能
- **振幅・位相の個別制御**: 各サブキャリアの振幅（0〜1）と位相（0〜360度）を調整
- **リアルタイム可視化**:
  - 周波数領域：各サブキャリアの振幅表示
  - 時間領域：IDFT後の合成信号（実部・虚部）
  - 個別波形：各サブキャリアの時間領域波形

## 使い方

### 操作方法

1. **サブキャリア数を調整**
   - スライダーで4〜16個の範囲で設定
   - サブキャリア数が増えると、より細かい周波数分割が可能に

2. **各サブキャリアの設定**
   - 振幅：0（送信なし）〜1（最大）
   - 位相：0〜360度
   - リアルタイムで波形が更新されます

3. **波形を観察**
   - 周波数領域グラフ：各サブキャリアに割り当てられた振幅
   - 時間領域グラフ：IDFT後の実部（青）と虚部（緑）
   - 個別波形グラフ：各サブキャリアの波形を重ねて表示

### 実験のヒント

- **直交性の確認**: サブキャリア0だけ振幅1、他を0にして波形を観察
- **位相変化の影響**: 同じ振幅で位相を変えると、時間領域信号がどう変化するか
- **重ね合わせの原理**: 複数のサブキャリアを有効にして、合成信号を観察
- **QPSK/16QAMのシミュレーション**: 振幅と位相を組み合わせて、実際の変調方式を模擬

## 技術詳細

### IDFT（逆離散フーリエ変換）

周波数領域の複素数値 X[k] を時間領域の信号 x[n] に変換します：

```
x[n] = (1/N) Σ X[k] * e^(j2πkn/N)
     = (1/N) Σ X[k] * (cos(2πkn/N) + j*sin(2πkn/N))
```

- N: サンプル数（このデモでは64固定）
- k: サブキャリアインデックス（0 〜 N-1）
- n: 時間サンプルインデックス（0 〜 N-1）

### サブキャリアの直交性

サブキャリアkとk'が直交する条件：

```
Σ e^(j2π(k-k')n/N) = 0  (k ≠ k')
```

これにより、各サブキャリアは互いに干渉することなく、同一周波数帯域を共有できます。

### 実装

- **フレームワーク**: Vanilla JavaScript（依存関係なし）
- **グラフ描画**: Chart.js 4.4.1
- **サンプル数**: 64（固定）
- **最大サブキャリア数**: 16
